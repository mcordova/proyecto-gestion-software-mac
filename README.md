# proyecto-gestion-software-mac
Ejemplo para el curso de Gestión de Proyectos de Desarrollo de Software

## Nombre del proyecto
Proyecto Gestion Software MAC

## Objetivo
Apoyar el contenido teórico del curso para recordar las contraseñas, con un comprobador, un "tester" que permita validar los criterios enseñados en los contenidos del curso.


## ¿Qué problema resuelve?
Una vez elegida una contraseña... comprobar que se cumplen los criterios de la teoría explicada en el curso del Método Infalible para Acordarse de las Contraseñas

En la teoría se explica que la contraseña debe ser única, segura, fácil de recordar pero difícil de adivinar (difícil de romper)

## Tecnologías que podría usar
JavaScript
